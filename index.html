Система учета студентов

Самая дерзкая система для учета оценок, предметов и студентов всех времен.

Фичи:

  • Поиск по имени и оценке
  • Чтение из файлов
  • Сохранение в журнал
  • Смена языка

Пример кода

std::find_if(students.begin(), students.end(), [&](const std::string& s) {
    return s == nameStudent;
});
